General Issues

Why can't I connect bzShare?

First, you could not have asked this question if you see this in the first place.

  • The network connection is lost or interrupted. Please check your network connection and proxy settings before you try again.
  • Your web browser is too old to use bzShare.
  • The local firewall had prevented you from visiting bzShare. We cannot help and we do not offer to help these kinds of problems.

Why can't I use bzShare?

These are a set of problems related to the deprecation of obsolete web browsers. Consider updating your web browsers in order to fix these problems.

  • Your web browser does not support JavaScript or you turned it off.
  • Your web browser does not support CSS3.
  • Your web browser does not support HTML5.
  • You have an IE6. :)

Supported Browsers

Having at least these versions of browsers enable your enhanced experience of bzShare. Older versions have not been tested, but should at least work.

  • Mozilla Firefox 37.0+ Download

Files Related

Why can't I upload anything?

This might be a result of a great many reasons. If you cannot find the correct answer, please contact server administrator directly or consult the developer on GitHub.

  • The network connection is lost or interrupted.
  • You do not have the permissions to write to the current folder. There is no solution to this problem, and please upload the file somewhere else.
  • The file itself is too large to be uploaded. Please consider compressing it to smaller chunks with compression softwares (e.g. 7-Zip) and then upload to bzShare.

Why can't I open something in bzShare?

It might be a result of network latency. By default bzShare would not allow a user to see anything that he would not be allowed to see, in any way.

  • The network connection is lost or interrupted. Try refreshing the page to reset the webpage and then check if the open operation works.
  • Your web browser is too old to use bzShare. If you cannot decide what version to choose, please view the related sections of FAQ.
  • Your computer is too laggy to open the file. Try downloading this somewhere else, then check whether you are able to open this file.

Why can't I rename something in bzShare?

This is a combination of problems.

  • The network connection is lost or interrupted.
  • You do not have RW (Read / Write) permissions to this very file / folder.

Why can't I move or delete something in bzShare?

This is also a combination of problems.

  • The network connection is lost or interrupted.
  • You do not have full RW (Read / Write) permissions of this file / folder and its subcontents. You may not move or delete if you do not have these permissions, even though you can read them.

Previewing Related

Why can't I preview this file in bzShare?

This is a typical problem which can be invoked in bzShare.

  • bzShare had not yet been designed to implement a preview method for this type of file. In this case, you should download it to your device and view it elsewhere.
  • On iPhones, iPads and other iOS devices, you would be able to preview Office (*.doc, *.docx, *.ppt, *.pptx, *.xls, *.xlsx, etc.) by clicking 'Download' button. This directly saves the file to the temporary area where you will be able to preview it.
  • In the case that you are sure this is a type that bzShare can handle, consult other sections to see the details of the corresponding file type.

Why can't I preview this image in bzShare?

This is a typical problem which can be invoked in bzShare.

  • Your web browser do not meet the browser standards of bzShare.
  • The connection to the remote server has been interrupted.

Why can't I preview this PDF document in bzShare?

This is a typical problem which can be invoked in bzShare.

  • Check the diagnosis in 'Why can't I preview this image in bzShare' first.
  • The PDF file uploaded might be damaged or corrupted.

Why can't I preview this video in bzShare?

This is a typical problem which can be invoked in bzShare.

  • Check the diagnosis in 'Why can't I preview this PDF document in bzShare' first.
  • The video has an obstruse format that few people can decode.
  • The previewing of videos is related to the native support of the browser. Different browsers on different platforms may act differently to video previews, and since this differs greatly, we suggest that you give up previewing videos on bzShare, elsewise consult the documentation of the corresponding operating system and browser for further information that we cannot provide.

Account Related

Why can't I create an account?

There are a certain limitations on the parameters while creating the account.

  • To avoid alien intrusion and potential DDoS, unauthorized users' access of the bzShare platform, we implemented an 'invitation code' method, which would block unrelated users from gaining access to most of the bzShare functionalities. People who fail to enter the correct code upon registration would not be able to create the account.
  • The length of the user handle that is entered upon login, should not be shorter than 3 characters, and should not be longer than 32 characters. It should also be composed executively of non-capital letters and numerics only ([a-z0-9] as in RegEx).
  • The length of the password should not be shorter than 6 characters, and should not be longer than 64 characters. The password should only consist of letters that you can enter with a standard QWERTY / American-101 keyboard.
  • You are strongly advised against using common combinations of passwords, as well as passwords that you use in your everyday activities on other websites and services. By doing so you inadvertently expose your data and privacy to unbeknownst people, not excluding the server administrator. We will not take resposibilities for the potential threats and risks, as we explicitly stated.
  • The user name should not be shorter than 3 characters, and should not be longer than 32 characters. The user name is what would display on common areas, which could be constructed with arbitrary Unicode characters, excluding a handful of HTML escape characters.
  • The description takes similar restrictions as user names, except the length is only bounded by 128 characters merely.

Why can't I change my avatar image?

There are also a few limitations on the image.

  • The file you uploaded must be an image.
  • The size of the image should be smaller than 1MB and larger than 68 bytes.

Why can't I sign in?

This is an awkward question that you should never ask. Neither do you wish to ask this question.

  • The username you entered did not exist.
  • You have been banned by the server administrator for improper activities.
  • The password you had entered is incorrect.
  • Somebody else had changed your password elsewhere.
  • Literally, you had forgotten your password. Ask the server administrator for help on resetting your password by providing your identity information. This action should not be supposed to be done online.

Why am I banned?

Mostly, the server administrator does not ban you for no reason.

  • For safety of the server and the server administrator, any attempt to post and upload unlegitimate, illegal files and data to the server, would be rightfully rewarded for a constant time of ban.
  • You have uploaded something that had caused general dispution and chaos, damaging the reputation of the server and the server administrator, as well as something that considered offensive to particular groups of people.
  • You have abused your rights of using bzShare as a normal user, including spamming the server with excessively small files and intensively large files. In other cases you had overflowed the server with computationally-intensive tasks that occupied the server.
  • Most unlikely, the administrator just found banning people an interesting thing to do.

Usergroup Related

Why can't I create a usergroup?

There are a certain limitations on the parameters while creating the usergroup.

  • Excluding the server administrator, no account may create more than 3 usergroups.
  • The length of the usergroup handle, should not be shorter than 3 characters, and should not be longer than 32 characters. It should also be composed executively of non-capital letters and numerics only ([a-z0-9] as in RegEx).
  • The usergroup name should not be shorter than 3 characters, and should not be longer than 32 characters. The usergroup name is what would display on common areas, which could be constructed with arbitrary Unicode characters, excluding a handful of HTML escape characters.

Why can't I join a usergroup?

The reason is simple.

  • Either your network connection is lost or interrupted.
  • Or that usergroup didn't even exist.
  • In other cases you have already requested to join the usergroup.
  • Or you are already a member of that group.

Why can't I accept an account into my group?

If you are aware, this question should never have been asked.

  • Your network connection is lost or interrupted.
  • Maybe you aren't an owner of that group.
